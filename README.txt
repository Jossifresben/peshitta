Syriac New Testament dataset - traditional 22-book canon - UTF-8 Unicode Syriac

What this is
- A machine-readable dataset derived from the ETCBC/syrnt plain-text corpus.
- Canon scope: the traditional 22-book Syriac New Testament canon associated with the Peshitta tradition:
  Matthew, Mark, Luke, John, Acts, Romans, 1-2 Corinthians, Galatians, Ephesians, Philippians, Colossians,
  1-2 Thessalonians, 1-2 Timothy, Titus, Philemon, Hebrews, James, 1 Peter, 1 John.
- Encoding: UTF-8 Unicode Syriac.
- Rows/verses: 7440.

Important limitation
- This is NOT a diplomatic Unicode transcription of the Khabouris manuscript itself.
- It is a Unicode Syriac dataset built from ETCBC/syrnt and restricted to the 22-book traditional canon.
- Use this for search, NLP, indexing, alignment, and software work.
- If you need manuscript-specific Khabouris readings, those need to be extracted separately from Stephen P. Silver's Khabouris transcription.
